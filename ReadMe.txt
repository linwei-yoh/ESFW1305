版本号:

V1.0.0  不带循环正反转功能  程序采用老的格式

V2.0.0  有循环正反转功能    采用新的程序结构
   V2.0.1 修改了堵转报错功能 解决了堵转计时 在完成一次运行后可能没有清零的问题	
          修正了过零信号端口的上拉输入 改为 悬浮输入
   V2.0.2 修正了带缓冲的ADC 处理方式                                                           v2.0.3 改变了设定速度变化后 速度的跟随方式

V3.0.0
	增强程序稳定下 
	以及电机在低速情况下 减少震动的发生
	加入UID功能
	可控硅最大延迟触发时间改为8000
                                                                                             V3.0.1
	1.在反馈模式下 用面板按键修改速度可以显示当前设定速度
        2.按键修改速度时，会有1S的延时时间。在延时时间内附带闪烁效果
          闪烁完成后，才执行速度设定 和保存功能
        3.修正了 修改速度边界时，当前速度可能出界的问题
        4.按键模式下 修改速度时，仅当电机没有在运行时才执行一次E2Rom保存

V3.0.2
	1.去除掉速度修改时候的闪烁确定
	2.将EEPROM保存 全限制为电机停止时才保存
	3.速度的保存延时改为1.5S
	4.改变旋钮方式下，获得的设定速度 对其进行四舍五入           

V3.0.3
	修改滤波初始值为0XFF
	将停止时的判断从Motor.SpdCvt.SpdTar 改为 Motor.SpdCvt.RefSpd
	反馈速度显示 才用独立参数与接口
	在缓慢停止模式下，停止时，如果再次出发运行信号，则立刻启动。
	

3.0.4
	外部输入改为 悬浮状态

V3.0.4.1  
	去除误导加密，修改正式加密的算法

3.0.5
	增加循环模式下 正转速度和反转速度设定参数
 
3.0.5.1
	去除误导加密，修改正式加密的算法
	速度缓冲池大小改为2

3.0.6
	使用原先的加密公式
	修复一个3000转时 宏定义上的BUG
	改进速度反馈运算
	速度缓冲池大小为8

3.0.6.1
	使用脱机下载器的加密公式

3.0.7
	优化速度捕获过程对 对干扰信号的过滤
	重新设置过零信号的获取方式，改为捕获+滤波

3.0.7.1
	使用脱机下载器的加密公式

3.0.8
	将IOCtrl中 关于导通角的控制相关函数移动到PwnGen中。
	将导通角延时时间改为局部变量。
3.0.8.1
	使用脱机下载器的加密公式	

3.0.9(独立版本 额外功能 可能存储空间不足)
	在速度控制方式选择中增加一项 根据正反转设定速度 控制正反转信号的执行   

3.1.0
	从3.0.8改来 修改了最大/最小导通角设定时间
                                                             
3.1.1
	将过零信号作为掉电检测 用于保存EEROM

3.1.2.1
	1、将速度反馈的滤波加大 
	2、将PID的P值改为3000  
	
3.1.3.1
	1、修复当F-03选择记忆面板运行状态时，修改速度不保存运行状态的BUG

3.2.0
	1.增加速度反馈滤波参数选项 (附带修改滤波参数从大到小时，运算溢出的错误
	2.修改EEPROM的保存方式，从字节编程改为字编程
	
3.2.1
 	1.脱机下载方式
